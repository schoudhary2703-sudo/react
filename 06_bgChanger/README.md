# 🎨 Background Color Changer

A lightweight React mini-project that allows users to dynamically change the application's background color. The project features a clean sidebar UI and demonstrates core React concepts like state management and list rendering.

## ✨ Features

* **Dynamic State Updates:** Instantly changes the background color of the main window using React's `useState` hook.
* **Component Mapping:** Uses the JavaScript `.map()` array method to cleanly and dynamically render the sidebar color buttons, keeping the code DRY (Don't Repeat Yourself).
* **Sidebar Layout:** Implements a modern `aside` sidebar configuration for the user interface.
* **Quick Reset:** Includes a dedicated action to revert the canvas back to its default state.

## 🛠️ Technologies Used

* React.js
* Tailwind CSS (for layout and styling)
* HTML / JSX
