## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/schoudhary2703-sudo/react.git
```

### Move into a project folder

Example:

```bash
cd 04_tailwind
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

Open the local development URL shown in your terminal.
