# ⚛️ React Learning Journey 🚀

A collection of React projects created while learning React.js from basics to advanced concepts.

This repository contains practical examples covering React fundamentals, custom rendering, state management, Virtual DOM, props, and Tailwind CSS integration.

---

## 📌 Repository Overview

This repository is organized chapter-wise, where each folder represents a React concept or mini project.

---

## 📂 Projects Structure

```bash
react/
│
├── 01_Basic/
│   └── React basics and library fundamentals
│
├── 02_customReact/
│   └── Understanding custom rendering using JavaScript and HTML
│
├── 03_counterProject/
│   └── Counter application and state management concepts
│
├── 04_virtualDOM/
│   └── Virtual DOM concepts and Tailwind CSS introduction

└── README.md
```

# 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/schoudhary2703-sudo/react.git
```

Go inside any project folder:

Example:

```bash
cd 05_propsAndTailwind
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
